# calendar
这个我一般当做日历插件模板，可根据需求随便改成想要的日历插件；


## Function
```javascript
var calendar = new Calendar('#demoDate', {
	// 最小限制日期（参数可选）
	minDate: '2017-02-11',

	// 最大限制日期（参数可选）
	maxDate: '2017-05-11',

	// 回调（参数可选）
	callBack: function (nowVal) {
		//nowVal为当前选择的值
	}
});
```
